# Day 1: Setting Up My Cybersecurity Lab 🖥️

## Introduction
Today marks the beginning of my cybersecurity journey. I set up my personal cybersecurity lab to practice hands-on skills throughout the month.

---

## What is a Cybersecurity Lab?
A cybersecurity lab is a safe environment where you can practice security tools, run experiments, and learn without affecting real systems.

---

## Tools I Used
- VirtualBox
- Kali Linux

---

## Why I Chose Kali Linux
Kali Linux is a popular operating system used by cybersecurity professionals. It comes pre-installed with many security tools used for testing and analysis.

---

## Setup Process
Here are the steps I followed:

1. Downloaded VirtualBox
2. Downloaded Kali Linux VirtualBox image
3. Extracted the downloaded file
4. Imported the Kali Linux machine into VirtualBox
5. Started the virtual machine
6. Logged in using default credentials

---

## Challenges I Faced

- I had to spend hours on youtube to learn how to properly setup my virtualbox
