## What is Linux?
Linux is an open-source operating system widely used by cybersecurity professionals. It is known for its flexibility, security, and powerful command-line interface.

---

## Commands I Learned

- `pwd` → Displays the current directory
- `ls` → Lists files and directories
- `cd` → Changes directory
- `cd ..` → Moves back one directory
- `mkdir` → Creates a new directory
- `touch` → Creates a new file
- `rm` → Deletes files
- `rm -r` → Deletes directories
