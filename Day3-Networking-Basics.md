# Day 3: Networking Basics 🌐

## Introduction
On Day 3 of my cybersecurity journey, I learned the basics of networking. Understanding how devices communicate is essential for identifying and preventing security threats.

---

## Key Concepts

### IP Address
An IP address is a unique identifier assigned to devices on a network.

---

### DNS
The Domain Name System (DNS) translates domain names into IP addresses.

---

### HTTP vs HTTPS
HTTP is not secure, while HTTPS encrypts communication for better security.

---

### Router
A router connects devices to the internet and directs network traffic.

---

## Practical Commands

- `ip a` → Displays IP address
- `ping google.com` → Tests connectivity
- `ip route` → Shows network routes
